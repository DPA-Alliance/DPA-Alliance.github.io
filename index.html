Data Process Automation - Consortium 소개

Data Process Automation Product를 구성하기 위해 모인 업체들의 컨소시엄입니다.
기존 데이터 처리 프로세스를 개선하여 데이터 처리에 드는 비용과 시간을 절감시키는 것이 목적이며
이를 통해 데이터를 통해 부가가치를 창출하는 기업들의 비즈니스 파트너가 되고자 노력하는 중입니다.
현재 각 분야의 전문가들이 모여서 Product를 패키징 및 개선하고 있습니다.

데이터는 21세기의 석유

최근 들어 데이터 분야에 대한 지원과 관심이 증대되고 있으며, 향후 4차 산업혁명시대의 신성장동력으로 주목을 받고 있습니다.

데이터는 고갈되지 않는다는 특징이 있으며, 오히려 분석과 사용을 할 수록 그 가치가 떨어지지 않고 더 증가하는 경향이 있습니다.

데이터 처리 인프라를 갖추기 위해 초기 투자비용이 많이 든다는 문제만 해소되면, 데이터를 활용해 '지속적인 부가가치 창출'이 가능해집니다.

현 데이터 분석 프로세스의 한계

보편적인 데이터 처리 프로세스는 각 단계별로 분업화가 되어있다는 특징이 있습니다, 그리고 각 단계에서 데이터 처리에 참가하는 구성원들이 제각기 다른 툴을 사용하기 때문에 데이터 전처리 및 통합에 지나친 비용이 발생하는 추세입니다.

또한 데이터 분석 프로세스를 진행하기 위해 각 단계마다 별개의 프로그램, 파일 포맷을 다루기 때문에 유지보수가 어렵고, 신규 직원의 교육에도 상당한 비용이 발생합니다.

특히 기존의 데이터 전처리 및 데이터 분석 툴은 Python, R등의 프로그래밍 언어 코드로 이뤄진 경우가 많아 신규 인력 교육 및 수급, 유지 보수의 어려움이 한층 더 배가됩니다.

상기한 문제점들은 데이터 처리 과정에서 단계별로 각기 다른 프로그램을 사용하였기에 발생한 문제점이며, 위에서 언급한 모든 과정을 한 가지 툴에서 진행할 수 있다면 해결될 것들입니다.

End to End Data Process Automation 솔루션을 통한 문제해결
데이터 전처리 및 통합 비용 절감

DPA는 데이터 처리 프로세스의 시작인 데이터 수집부터, 가장 마지막 단계인 데이터 분석 & 저장까지 동일한 솔루션 안에서 진행하여 기존 프로세스에서 낭비되던 경제적 비용을 절감하고 절약된 경영 자원 들을 실질적인 데이터 분석 및 부가가치 창출에 집중시키는데 도움이 됩니다.

또한 데이터 처리를 하기 위해 각 단계의 프로그램을 일일히 익힐 필요가 없고
GUI환경을 제공하는 솔루션 Tool하나만 숙지하면 되기에 직원 교육 및 유지 보수에 드는 비용 또한 절감됩니다.
